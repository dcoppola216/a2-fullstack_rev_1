What does "Full" mean in Full Stack Mean?

Full means that you can write code for back end, for the server, and front end, what the client is seeing on their own computer screen.

What does "Stack" mean in Full Stack Mean?

Stack means that many programs are used to create a website. Programs like HTML5, CSS3, and JavaScript are used to create what you see on screen. PHP and MySQL are used for the server.

What does "LAMP Stack" mean?

LAMP stands for:

L=Linux, which is the basis of our server operating system

A=Apache, is our web server

M=MySQL, our database

P=PHP is the code language we use on the server side

1."Databases"

Databases are a way to save data on the server. They are how businesses track orders, take order, keep track of inventory, and customers.

2."MySQL"

MySQL is a type of database and the second most used in the world.

3."SQL"

SQL stands for structured query language. It is the language that you write the code. This language work in MySQL, Oracle SQL, Microsoft SQL, and others.

4."PHP"

PHP is the language of code that conects the front end (your computer) to the back end (the server).
